# 🩺 Flutter LepraCheck App

Aplikasi mobile berbasis Flutter untuk membantu **deteksi dini penyakit kusta (Leprosy / Hansen’s Disease)** menggunakan teknologi kecerdasan buatan (Machine Learning).

---

## 📌 Deskripsi

Flutter LepraCheck adalah aplikasi yang dikembangkan untuk **keperluan penelitian dan pendidikan** dalam bidang teknologi kesehatan.  
Aplikasi ini memungkinkan pengguna untuk:
- Mengambil gambar kulit melalui kamera
- Mengunggah gambar dari galeri
- Melakukan klasifikasi apakah terdapat indikasi penyakit kusta atau tidak

Aplikasi ini bertujuan untuk mengeksplorasi penerapan Machine Learning dalam mendukung deteksi awal penyakit kulit, khususnya kusta.

---

## 🎯 Tujuan

- Mendukung kegiatan penelitian dalam bidang kecerdasan buatan dan kesehatan
- Menjadi media pembelajaran dalam pengembangan aplikasi berbasis AI
- Mengimplementasikan teknologi Machine Learning pada platform mobile
- Memberikan gambaran awal deteksi penyakit kusta secara digital

---

## 🚀 Fitur Utama

- 📷 Ambil gambar dari kamera
- 🖼️ Upload gambar dari galeri
- 🤖 Klasifikasi gambar menggunakan model Machine Learning
- 📊 Menampilkan hasil deteksi (Kusta / Bukan Kusta)

---

## ⚠️ Disclaimer

Aplikasi ini dikembangkan **hanya untuk keperluan penelitian dan pendidikan**.  
Hasil yang diberikan **bukan merupakan diagnosis medis resmi** dan tidak dapat dijadikan sebagai dasar pengambilan keputusan medis.  

Pengguna tetap disarankan untuk berkonsultasi dengan tenaga medis profesional untuk diagnosis yang akurat.
